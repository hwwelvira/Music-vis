# 音乐流派与声学特征可视化系统

这是一个基于 React、Vite 和 ECharts 构建的交互式音乐数据可视化项目。系统围绕音乐流派结构、声学特征、情感分类、歌曲分布和歌名词汇等信息进行多视图分析。

主要功能包括：

- 使用双层旭日图展示大类流派与细分流派的层级和规模分布。
- 使用弦图和七维雷达图对比不同流派之间的关联与声学特征。
- 使用能量和愉悦度散点图展示歌曲情感分类，并支持框选联动。
- 支持跨流派声学特征和情感分布对比。
- 通过歌曲列表、单曲详情、流派词云和流行度时间线进行进一步探索。

项目数据位于 `public/data.json`。

## 本地部署与运行

### 环境要求

- Node.js `20.19+` 或 `22.12+`
- npm

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后，在浏览器中打开终端显示的本地地址，通常为：

```text
http://localhost:5173/
```

### 构建生产版本

```bash
npm run build
```

构建结果将生成在 `dist` 目录中。

### 本地预览生产版本

```bash
npm run preview
```
